如何设置坐标系?

坐标系为顶层crs属性

{
    type: "FeatureCollection",
    features: [...],

    //crs 不存在则视为4326
    crs:{
        type:"name",
        properties:{
            name: "urn:ogc:def:crs:EPSG::3857",

            // 可以是proj或wkt字符串, 非4326或3857必须提供
            // 参考proj4.defs
            defs: ..., 
        }
    }
}
